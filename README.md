### How to Run
1. Clone this repo:
`$ git clone [repo-link]`
2. Create your environment (virtualenv):  
`$ virtualenv -p python3 venv`  
`$ source venv/bin/activate` (bash) or `$ venv\Scripts\activate` (windows)   
`$ (venv) cd [repo-name]`  
`$ (venv) pip install -e`  
    Or (conda):  
`$ conda env create -f environment.yml`
